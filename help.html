Your Java Friend - Help

Here is a list (guide) of all possible commands you can give to the system. You can try any of them by saying the word or words in the inverted commas ("...").

Greetings

You can greet the system with "hello", "hey", "hi" or "greetings".
You can also ask the system "how are you?".

Goodbye

You can say good bye to the system either with "bye" or "goodbye".

Open help

You can ask the system to open the help page by saying anything having the "help" keyword.

Command Not Recognised

If the system does not recognise your command, it will inform you about it. You can then try the command again.

Fun change name

You can ask the system to change its name by saying "change your name to 'name'". The name should be one word only.

Reset name

You can ask the system to reset its name to the default (Your Java Friend) by saying "reset your name".
If the name is already default, the system will tell you about it.

What's your name?

You can ask the system it's name by saying "what's your name?". It will either reply you with a funny text or its name.

Clear conversation

The system's conversation can be cleared by saying "clear the conversation", "delete all" or "delete messages". It will remove everything.
If you want to clear the conversation and have a help message, you can ask the system to "clear the conversation and start over again".

Koi Mil Gaya

You can have fun with the system by saying "koi mil gaya". It will respond with a funny message.

What is Java?

You can ask the system "what is java?", "java description" or "define java" for it to respond with the definition of Java.
If you want a broader definition of Java, you can ask "open java definition", "open what is java?" or "open java description".

Is Java OOP?

You can check whether Java is object oriented by asking "is java object oriented?".

Advantages of Java

You can learn about the advantages of Java by asking "why learn java", "why use java?" or "why do we use java?".
To learn more about the advantages of Java, you can ask to "open java advantages", "advantages of java", "open what are the advantages of java" or "open java benefits".

Downloads - NetBeans

You can download NetBeans by asking "download NetBeans".

Downloads - Java SE

You can download the Java SE/Development kit by asking "download java", "download development kit" or "download SE".
