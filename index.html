What is Java?

Java is a general-purpose, class-based, object-oriented programming language designed for having lesser implementation dependencies. It is a computing platform for application development. Java is fast, secure, and reliable, therefore. It is widely used for developing Java applications in laptops, data centers, game consoles, scientific supercomputers, cell phones, etc.

What is "Your Java Friend?"

"Your Java Friend" is an application built by LaunchTech to help students of Middlesex University Mauritius to learn and understand the programming language of Java either by typing commands or by talking to the application. When using "Your Java Friend", you will be able to learn different functions and syntaxes of Java easily in different sections.

How will it help you?

"Your Java Friend" will help and guide a student to learn, practice and build applications in Java. Students will be able to communicate to the system using their voice. This will help students learn how to write specific syntaxes in case they do not.
